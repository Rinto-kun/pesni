# 46. ИСТИНА ЧУДНА  
  
ИСТИНА ЧУДНА ЕДНА  
ИМА ВЪВ БИБЛИЯТА:  
"ВСИЧКО СЪДЕЙСТВА  
ЗА ТВОЙТО ДОБРО,  
ЩОМ СИ ВЪЗЛЮБИЛ ХРИСТА!"  
  
ГРИЖИ, ПРОБЛЕМИ В СВЕТА,  
МОЖЕ ДА ИМАШ, НО ЗНАЙ:  
"ВСИЧКО СЪДЕЙСТВА  
ЗА ТВОЙТО ДОБРО,  
ЩОМ СИ ВЪЗЛЮБИЛ ХРИСТА!"  
  
АКО ЗА НЕЩО В СВЕТА  
МИСЛИШ, ЧЕ НЯМАШ УСПЕХ,  
ЗНАЙ:  
"ВСИЧКО СЪДЕЙСТВА  
ЗА ТВОЙТО ДОБРО,  
ЩОМ СИ ВЪЗЛЮБИЛ ХРИСТА!"
