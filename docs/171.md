# 171. АЗ ИМАМ СЪЗДАТЕЛ

ТОЙ НАПРАВИ СЪРЦЕТО МИ  
ПРЕДИ СЪТВОРЕНИЕТО,  
АЗ СЪМ БИЛ В РЪЦЕТЕ МУ!

  ТОЙ ЗНАЕ МИСЛИТЕ МИ
