# 142. НЕЖНО ТИ МИНАВАШ  
  
НЕЖНО ТИ МИНАВАШ  
ПОКРАЙ МЕН,  
ЧУВСТВАМ ТЕ СЕГА ДО МЕН.  
НЕЖНО ГАЛИШ МЕ С РЪКА,  
ТИ РАЗТАПЯШ МОЯТА ДУША.  
  
ИСУСЕ, ИСУСЕ,  
ДЕН И НОЩ СИ ТИ ДО МЕН.  
КАК ДА ТИ СЕ ОТПЛАТЯ -  
ОСВЕН ЖИВОТА СИ  
ДА ТИ ДАРЯ.  
  
СЛЪНЦЕ СИ В ЖИВОТА МИ,  
ТИ ОГРЯВАШ ПЪТИЩАТА МИ.  
СЛЪНЦЕ СИ В ЖИВОТА МИ,  
БЕЗ ТЕБ НЕ МОГА, ГОСПОДИ!
