# 207. СВЯТИ ДУХ  
  
СВЯТИ ДУХ, ТЕБЕ ТЪРСЯ АЗ.  
ГОВОРИШ МИ  
СЪС ТВОЯ НЕЖЕН ГЛАС.  
  
НЕЖЕН СИ ТИ, О, СВЯТИ ДУШЕ,  
ТИ ДА ПРОМЕНЯШ  
СЪРЦАТА НАШИ.  
НЕЖЕН СИ ТИ, О, СВЯТИ ДУШЕ,  
ТИ ДА ПРОМЕНЯШ  
НАШИТЕ СЪРЦА.  
  
СВЯТИ ДУХ,  
АЗ ЩЕ ВЪРВЯ СЛЕД ТЕБ  
ПОКАЖИ МИ ТИ  
ТВОЯ ПЛАН ЗА МЕН.  
  
СВЯТИ ДУХ, АЗ ТЕ ОБИЧАМ  
ИЗПЪЛНИ МОЯТА ДУША.  
  
СИЛЕН СИ ТИ, О, СВЯТИ ДУШЕ,  
ТИ ДА ПРОМЕНЯШ  
СЪРЦАТА НАШИ.  
СИЛЕН СИ ТИ, О, СВЯТИ ДУШЕ,  
ТИ ДА ПРОМЕНЯШ  
НАШИТЕ СЪРЦА.
