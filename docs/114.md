# 114. ТИ СИ МОЯТ ИЗЦЕРИТЕЛ

„АЗ СЪМ ТВОЯТ БОГ, КОЙТО ТЕ ЦЕРЯ,  
АЗ СЪМ ГОСПОД, ТВОЙ СПАСИТЕЛ!

АЗ СЪМ ТВОЯТ ИЗЦЕРИТЕЛ!”  

ТИ СИ МОЯТ БОГ, КОЙТО МЕ ЦЕРИШ,

ТИ СИ МОЯТ ИЗБЕРИТЕЛ!
